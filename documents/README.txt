Place your PDF documents in this folder. The website 
references the following file names:

  1. cv-ruyel.pdf          → Your Curriculum Vitae
  2. transcripts.pdf       → Academic Transcripts
  3. certifications.pdf    → Professional Certifications
  4. technical-report.pdf  → Sample Technical Report
  5. research-paper.pdf    → Published Research Paper
  6. recommendations.pdf   → Letters of Recommendation

To add more documents:
  1. Place the PDF file in this folder
  2. Open index.html
  3. Add a new <div class="doc-card"> block in the 
     Documents section, following the same pattern 
     as existing cards
  4. Update the href to point to your file:
     href="documents/your-file.pdf"
